Test VIIIb

Úkolem je realizovat funkci (ne celý program, pouze funkci), která dokáže vyhledat v zadané čtvercové 2D matici celých čísel diagonálu s největším součtem. Přitom prohledává pouze diagonály rovnoběžné s hlavní diagonálou (nevyhodnocuje součet diagonály vedlejší a diagonál rovnoběžných s vedlejší diagonálou).

Požadovaná funkce má rozhraní:

int maxDiagonal ( int ** mat, int size, int * maxSum );
  • mat je prohledávaná matice, předaná volajícím. Funkce může hodnoty v této matici číst, nesmí je modifikovat.
  • size je počet řádek a zároveň i sloupců v matici (matice je čtvercová).
  • maxSum je výstupní parametr, ve kterém funkce předá volajícímu nalezený maximální součet hodnot prvků na diagonále.
  • Návratová hodnota signalizuje úspěch / neúspěch hledání. Funkce musí vrátit hodnotu 1, pokud uspěje, hodnotu 0 pokud neuspěje. Za neúspěch považujte pokud maximálního součtu dosáhne více než jedna diagonála v matici. Pokud funkce uspěje, musí správně nastavit výstupní parametr maxSum. Pokud neuspěje, nemusí (nemá) výstupní parametr maxSum ovlivňovat.

Odevzdávejte zdrojový kód, který obsahuje Vaši implementaci funkce maxDiagonal. Do odevzdávaného souboru samozřejmě patří ještě další Vaše funkce, které jsou z funkce maxDiagonal volané. Naopak, v odevzdávaném souboru nesmí být vkládání hlavičkových souborů a funkce main (pokud vkládání hlavičkových souborů a funkci main zabalíte do bloku podmíněného překladu, mohou zůstat). Za základ řešení použijte ukázku níže.
