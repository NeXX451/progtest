Test VIIIa

Úkolem je realizovat program, který pro zadanou sekvenci celých čísel na vstupu určí kolik obsahuje podsekvencí lichých (nepárne, odd) a kolik podsekvencí sudých (párne, even) čísel.

Vstupem programu je kladné číslo n, které udává počet zpracovávaných čísel. Za ním následuje zadání n celých čísel. Program takto zadaná čísla načítá a průběžně zpracovává. Vstup lze chápat také tak, že se střídají posloupnosti sudých čísel a posloupnosti lichých čísel (za posloupnost lze považovat 1, 2, 3, ... po sobě jdoucích buď sudých nebo lichých čísel). Úkolem programu je určit počet těchto posloupností, zvlášť pro sudá a lichá čísla.

Výstupem programu jsou nalezené počty posloupností, podle minulého odstavce. Formát výstupu je zřejmý z ukázky.

Program musí ošetřovat nesprávná vstupní data. Pokud je na vstupu nečíselná hodnota nebo číslo n je záporné nebo nulové, program zobrazí chybové hlášení a ukončí se (viz ukázka).

Program je testován v omezeném prostředí, je limitován dostupnou pamětí i dobou běhu. Daná úloha není náročná na dobu běhu. Vstup není potřeba ukládat do paměti, stačí jej průběžně zpracovávat. Pokud si přesto budete vstupní data ukládat do pole, neprojde Váš program testem na efektivní využívání paměti (nedosáhnete plného hodnocení).
